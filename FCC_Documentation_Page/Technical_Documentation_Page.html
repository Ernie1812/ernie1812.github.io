Introduction

The Flexible Box Module, usually referred to as flexbox, was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.


When we describe flexbox as being one dimensional we are describing the fact that flexbox deals with layout in one dimension at a time — either as a row or as a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

Flexbox Layout

The two main components of a Flexbox layout are the container and the items. This example contains a container with three items:

<nav class="container">
<div>Home</div>
<div>Search</div>
<div>Logout</div>
</nav>

Before using the Flexbox layout, the <div> elements will be stacked on top of each other like this (some styling has been added for clarity):

To turn this into a Flexbox layout, give the container the following CSS property:

.container {
display: flex;
}

This will automatically position the items along the horizontal axis.

Justify content and Align items

Justify-content and align-items are two CSS properties which help us distribute the items in the container. They control how the items are positioned along the main axis and cross axis. In this case (but not always) the main axis is horizontal and the cross axis is vertical:

Center all the items along the main axis by using justify-content:

.container {
display: flex;
justify-content: center;
}

Or set it to space-between to add space between the items:

.container {
display: flex;
justify-content: space-between;
}

These are the values you can set for justify-content:

  • flex-start (default)
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

Controlling a single item

We can also control single items. If for example you want to keep the first two items on the left side, but move the logout button to the right side you can use the technique of setting the margin to auto.

.logout {
margin-left: auto;
}

If you want both the search item and the logout item to be pushed to the right hand side, add the margin-left to the search item instead.

.search {
margin-left: auto;
}

This will push the search item as far to the right hand side as possible, which will push the logout item with it:

The flex property

To create responsive items use the flex property. You can use this instead of percentages by targeting all the items and giving them a flex value of 1.

This stretches the items to fill the entire container.


In many cases you’d probably want one of the items to take up the extra width, and thereby only set one if them to have flexible width. You can for example make the search item take up all the extra space:

.search {
flex: 1;
}

The flex property is actually a shorthand three properties: flex-grow, flex-shrink and flex-basis. Together they allow us to control how flex items behave inside their parent container:

  • flex-grow determines how flex items grow inside their parent. Default 0
  • flex-shrink determines how flex items shrink when their parent gets smaller. Default 1.
  • flex-basisis determines the size of flex items before space is distributed and how content is treated. Default auto.
